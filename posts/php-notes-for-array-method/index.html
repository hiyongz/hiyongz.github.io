2. 检查数组或者变量方法

变量检查：

  • isset() - 检测变量是否已声明并且其值不为 null，释放变量使用 unset() 方法。
  • defined() - 检查某个名称的常量是否存在
  • empty() - 检查一个变量是否为空，PHP中的“空”元素包括：空字符串""、数字0 、浮点数0.0 、字符串"0" 、null、false、空数组array() 、声明了但没有值的变量$var。
  • is_null() - 检测变量是否为 null。
  • property_exists() - 检查对象或类是否具有该属性。
  • method_exists() - 检查类的方法是否存在。

数组检查：

  • array_keys($array) - 返回数组中所有键名。
  • array_keys($array, $search_value): 返回search_value 值的键名。
  • array_key_exists( $key, $array) - 检查数组里是否有指定的键名或索引。
  • array_search($needle, $haystack) - 在数组 $haystack 中搜索给定的值$needle，返回相应的键名。

3. 过滤数组元素

array_filter(array $array, ?callable $callback = null, int $mode = 0): array

  • array 数组中的每个值会传递给 callback 回调函数，如果 callback 回调函数返回 true，则将 array 数组中的当前值返回到结果 array 数组中。
  • 如果不设置 callback 回调函数，会删除数组 array 中的所有“空”元素。
  • 过滤后的array 数组键名不会重新索引，可以使用 array_values() 函数对数组进行重新索引。
  • $mode = 0 - 默认值，接受值作为唯一参数
  • $mode = ARRAY_FILTER_USE_KEY - 接受键名作为唯一参数
  • $mode = ARRAY_FILTER_USE_BOTH - 同时接受键名和键值
