进入Manage Jenkins，提示新版本，点击下载 jenkins.war 包。

或者进入官网https://www.jenkins.io/download/下载指定版本的war 包。

可能原因是jenkins.war只有只读权限，需要给它添加权限。而此时jenkins容器没有启动成功，我们是无法使用命令 docker exec -it -u root jenkins bash 进入容器内进行操作的。
