PHP环境部署

PHP适用于Web应用程序开发，这里介绍使用XAMPP来配置开发环境，它是一个免费的Apache发行版，包含了MariaDB、PHP和Perl。

由于用不到数据库，本文不介绍数据库安装和配置方法。
