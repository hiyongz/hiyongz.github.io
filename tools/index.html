  1. Github pwc
    以相当简洁的列表关联论文和代码实现，包含最新的 NIPS 论文。

    自动把论文连接到实现代码的 GitHub 资源库和数据集，并根据 GitHub 的收藏量排序。每篇论文可能有多个合并的条目。
